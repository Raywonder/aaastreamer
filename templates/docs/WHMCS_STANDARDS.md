# WHMCS Standards
Last updated: 2026-04-03

## Separation rules
Always separate:
- hosting
- domains
- app-management billing
- custom services
- split-payment work
- travel or project costs when billed through client work

## Installatron billing ladder
- Basic: $5/month for 1 app
- Standard: $10/month for up to 2 apps
- Pro: $20/month for up to 5 apps
- Unlimited: $50/month for unlimited apps

## Native UI safety
Use native WHMCS UI for one-off corrections, manual transactions, and edge cases when that is safer than automation.
